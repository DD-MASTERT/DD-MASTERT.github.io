快捷键可以通过不同的方式表示，包括使用特殊字符和大括号。以下是一些常见的快捷键表示方法和它们对应的参数使用说明：

快捷键 表示方法 参数说明

Tab {TAB} 制表键
Esc {ESC} 退出键
Space {SPACE} 空格键
Backspace {BS} 或 {BACKSPACE} 退格键
Delete {DELETE} 或 {DEL} 删除键
Home {HOME} 起始键
End {END} 结束键
Page Up {PGUP} 向上翻页键
Page Down {PGDN} 向下翻页键

Up {UP} 上箭头键
Right {RIGHT} 右箭头键

F1-F12 {F1} - {F12} 功能键 F1 到 F12

Num Lock {NUMLOCK} 数字锁定键
Caps Lock {CAPSLOCK} 大写锁定键
Scroll Lock {SCROLLLOCK} 滚动锁定键
Print Screen {PRTSC} 打印屏幕键

  • Ctrl + C："^c"
  • Alt + F4："%{F4}"
  • Ctrl + Alt + Delete："^{%}{DELETE}"
  • Ctrl + Shift + Esc："^{+}{ESC}"

注意，对于组合键：

  • ^ 表示 Ctrl 键。

  • % 表示 Alt 键。

  • + 表示 Shift 键，但通常只有当需要同时按下多个修饰键时才使用 + 号，例如 Ctrl + Shift。

  • Ctrl + A："^a"

  • Ctrl + B："^b"

  • Ctrl + C："^c"

  • Ctrl + V："^v"

  • Ctrl + X："^x"

  • Ctrl + Z："^z"

  • Ctrl + Y："^y"

  • Alt + Tab："%{TAB}"

  • Alt + Enter："%{ENTER}"

  • Ctrl + Shift + T："^{+}t"

至于参数使用，直接替换命令的参数位置即可，参数是由调用指令传递的

例如：打开网站，参数1
参数1对应脚本指令的第一个参数，最对可指定3个参数，依次类推
完整脚本指令格式为：[-脚本文件名-参数1-参数2-参数3-]
参数是可选的，如果没有参数可以省略。例：[-脚本文件名-]
只指定一个参数为：[-脚本文件名-参数1-]

下面是一个简单的示例：

假如我有一个脚本order.txt,内容如下：

 打开网站，参数1

现在我要用这个脚本打开网站https://www.bilibili.com/
该脚本调用指令就是：

 [-order-https://www.bilibili.com/-]
